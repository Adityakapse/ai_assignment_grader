AsymQ1.java is a Java program that calculates the asymptotic worst-case running time of two functions, one using a loop and another using a single array access. The program uses Thea notation to specify loops based on their length, as well as analyis with input text fields for both functions.

Method f() implements a standard algorithm for finding the sum of a given sequence of odd numbers in ascending order by repeatedly multiplying every third term by an arbitrary factor (in this case, 2). The function uses a nested loop over each index of the array `a`. The output analysi describes the number of iterations and the cost per iteration, which are both Theta(1) with respect to the length N of the input array.

Method g() implements the same algorithm using a single array access instead of a nested loop. As this function is only used as an example in the analysis, no further analysis or results are provided.

The program uses two points to specify loop iterations (valid range: 0–8), total loop cost (valid range: 0–6), and worst-case runtime for both functions. The overall worst-case runtime of f is accounted for by the nested loop part, while g is calculated using a single array access.
