Calculate the asymptotic worst case running time of the functions below, as a function of the length N of the input array `a`.

- Each step of the analysis should be justified with one or two sentences.

public final class AsymQ1 {
    // `a` has length N.
    // Your analysis should be based on N.
    public static int f(int[] a) {
        int n = a.length;
        int s = 0;
        for (int i = 0; i < n; i++) {
            for (int j = 0; j <= i; j++) {
                s += a[j] * a[i];
            }
        }
        int t = g(a);
        return s + t;
    }

    private static int g(int[] a) {
        int n = a.length;
        int t = 0;
        int k = n;
        while (k > 0) {
            t += a[k - 1];
            k = k / 2;
        }
        return t;
    }
}

SAMPLE SOLUTION:
Let N be the length of array a.

Method f
Part A: int n = a.length; int s = 0; — constant-time initialisation.
Part B (nested loops over i, j): the body executes for every (i, j) with 0 <= j <= i < N.
Part C: int t = g(a); return s + t; — a single call to g plus a constant-time return.

Parts A and the return statement of Part C are Theta(1).

Part B (nested loops):
- Combined iterations: when i = 0 the inner loop runs once; when i = 1 it runs twice; ...; when i = N-1 it runs N times. Total = 1 + 2 + ... + N = N(N+1)/2 = Theta(N^2).
- Cost per iteration: Theta(1) — two array reads, one multiplication, one += assignment, loop checks and increments.
- Total cost of Part B: Theta(N^2) iterations x Theta(1) = Theta(N^2).

Part C:
- Cost of the call to g is Theta(log N) (see analysis of g below); the addition and return are Theta(1).
- Total cost of Part C: Theta(log N).

Overall running time of f:
Theta(1) (Part A) + Theta(N^2) (Part B) + Theta(log N) (Part C) = Theta(N^2).

Method g
Part D: int n = a.length; int t = 0; int k = n; — constant-time initialisations.
Part E (while loop): each iteration accesses a[k-1] and halves k.
Part F: return t; — constant time.

Parts D and F are Theta(1).

Part E:
- Iterations: k starts at N and is replaced by k/2 each iteration, so the loop terminates after Theta(log N) iterations (k reaches 0 when it has been halved log2(N)+1 times).
- Cost per iteration: Theta(1) — one array read, one += assignment, one division, one comparison.
- Total cost of Part E: Theta(log N) x Theta(1) = Theta(log N).

Overall running time of g:
Theta(1) + Theta(log N) + Theta(1) = Theta(log N).

STUDENT SOLUTION:
The function f has a loop, so it runs in O(N) time. Inside the loop body it performs some small operations like adding and multiplying values from the array. The g method does some halving thing and looks small. The answer is f = O(N) and g = O(1).
