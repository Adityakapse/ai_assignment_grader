Yes, here is a function that calculates the Least Common Multiple (LCM):
```crystal
def lcm(first_num: int, second_num: int)
  gcd(first_num, second_num).to_f / gcd(gcd(first_num, first_num), second_num)
end
```
This implementation uses the GCD function from the lecture and implements it recursively. The recursion terminates when the GCD of the two numbers is 0 (or their remainder is 0). If there's a division by zero or no remainder, the LCM returns a non-zero value.
