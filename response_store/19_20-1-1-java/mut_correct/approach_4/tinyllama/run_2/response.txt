The GCD implementation from the lecture can be implemented as follows:
```java
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner input = new Scanner(System.in);
        
        System.out.print("Enter first number (int): ");
        int num1 = input.nextInt();
        System.out.print("Enter second number (int): ");
        int num2 = input.nextInt();
        
        int gcd = getGCD(num1, num2);
        
        if (gcd == 0) {
            System.out.println("Error: The given inputs do not have a GCD.");
        } else {
            System.out.println("LCM of " + num1 + " and " + num2 + " is " + getLcm(num1, num2) + ".");
        }
        
        input.close();
    }
    
    public static int getGCD(int a, int b) {
        if (a < b) {
            int tmp = a;
            a = b;
            b = tmp;
        }
        
        return gcd(a, b);
    }
    
    public static int getLcm(int a, int b) {
        return (a * b) / gcd(a, b);
    }
}
```

In this implementation, the function `getGCD()` is implemented to calculate the GCD between two given integers. The function returns the smaller of the two numbers as its output. The function `getLcm()` is then called with the inputs `num1` and `num2`, and it returns the LCM using the previously calculated GCD. This ensures that all input values are valid, and the output of the lcm function can be checked to ensure correctness.
