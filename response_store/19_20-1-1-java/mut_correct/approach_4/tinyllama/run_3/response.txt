Here's a Java implementation of the Least Common Multiple (LCM) function with the Greates Common Divisor (GCD) algorithm, which follows the standard method for finding the LCM:

```java
import java.util.*;
public class LCM {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.print("Enter two positive integers (a and b): ");
        int a, b;
        while (!(a = sc.nextInt()) > 0 && !(b = sc.nextInt()) > 0) {
            System.out.println("Invalid input, please enter positive integers only.");
        }
        if (a == 0 || b == 0) {
            System.out.print("LCM of " + a + "/" + b + " is not defined: base case reached.");
        } else {
            int lcm = gcd(a, b);
            System.out.println("LCM of " + a + "/" + b + " is:" + lcm);
        }
    }
    static int gcd(int a, int b) {
        if (b == 0) {
            return a;
        } else {
            int r = gcd(b, a % b);
            return r;
        }
    }
}
```

This implementation uses the GCD algorithm to calculate the LCM of two given integers. It first checks whether the inputs are valid (both positive integers) and prompts for an error message if they're not.
