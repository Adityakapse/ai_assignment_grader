1. Import the required module or package (`import math`) to use its built-in function `gcd`.

2. Convert the input values into integers using the `int()` function.

3. Call the "gcd()" function with two parameters, the first being the lower integer and the second being the second integer:

```python
import math

def gcd(a, b):
    if b == 0:
        return a
    else:
        g = gcd(b, a % b)
        return a // g * g

# example usage
a = int(input("Enter first integer: "))
b = int(input("Enter second integer: "))
gcd_result = gcd(a, b)
print("The greatest common divisor of", a, "and", b, "is", gcd_result)
```

this code will prompt the user for two integers and then calculate their gcd using the `gcd()` function with the provided input parameters. The output will show the calculated result.
