   def gcd(a, b):
       """
       Calculate the greatest common divisor (GCD) of two numbers a and b.
       
       Parameters:
       - a: the first number to be divided
       - b: the second number to be divided
       
       Returns:
       - The greatest common divisor (GCD)."""
       if b == 0:
           return a
       else:
           remainder = a % b
           return gcd(b, remainder)
    ```

   def lcm(a, b):

       Calculate the least common multiple (LCM) of two numbers a and b, recursively or iteratively.
       
       Parameters:
       - a: the first number to be divided
       - b: the second number to be divided
       
       Returns:
       - The least common multiple (LCM)."""
       if b == 0:
           return a
       else:
           remainder = gcd(b, a % b)
           return (a * remainder) / gcd(a, remainder)
    ```
