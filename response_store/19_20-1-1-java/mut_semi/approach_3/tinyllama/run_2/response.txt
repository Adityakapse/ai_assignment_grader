Here's a possible Java implementation of the Least Common Multiple (LCM) algorithm:

```java
import java.util.Arrays;

public class LeastCommonMultiple {
    public static int lcm(int a, int b) {
        return gcd(a, b) * Arrays.stream(new int[b]).sum();
    }
    
    private static int gcd(int a, int b) {
        if (a == 0) {
            return b;
        } else {
            return gcd(b, a % b);
        }
    }
}
```

To use this LCM algorithm, you'd call the `lcm` function with the two input integers. Here's an example usage:

```java
System.out.println(LeastCommonMultiple.lcm(21, 5)); // Output: 36
```
