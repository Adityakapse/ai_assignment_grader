To calculate the GCD of two given values, you can use the Euclidian algorithm in Python:

```python
def euclidean_gcd(a, b):
    """
    Returns the greatest common divisor of two integers, i.e., the largest integer n such that a * n = b. If either a or b is zero or one, returns 1. The algorithm is as follows:

    1. Divide a by b, if necessary to get an integer remainder r.
    2. If r == 0, then return a.
    3. Return the GCD of the two numbers a and b after performing step 2 for all common divisors b/r and a%b.
    """
    # Find the greatest common divisor (GCD) of the input integers.
    if a == 0:
        return b
    elif b == 1:
        return a
    
    # Calculate the remainder b//a, if any, and continue until no remainder is found.
    while b != 0:
        r = b % a
        if r == 0:
            break
        b //= r
        
    # Find the GCD of the two numbers a and b after performing step 2 for all common divisors b/r.
    return a if b == 1 else gcd(b, a % b)
```

To calculate the LCM (Least Common Multiple) of two integers, use the formula:

LCM = (a * b) / gcd(a, b)

where:
- A is the greatest common divisor (GCD) of the inputs.
- B is the LCM of the inputs.
- GCD is defined as the greatest integer less than or equal to both a and b.

Here's an example:

Suppose you have two integers, `a` = 6 and `b` = 2, so their GCD is 1. Now, suppose you want to calculate the LCM (Least Common Multiple) of these two integers. In Python, you can use the following code:

```python
import math

a, b = map(int, input().split())
lcm = (a * b) // gcd(a, b)
print("LCM =", lcm) # Outputs "LCM = 12"
```
