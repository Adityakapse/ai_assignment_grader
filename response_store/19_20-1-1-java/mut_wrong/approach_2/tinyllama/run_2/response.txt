Similar to the already known "Greatesst Common Divisor" (GCD), the Least Common Multiple (LCM) is also used in elementary mathematical applications, for example in the multiplication of heterogeneous fraction products or as a base point for certain planetary orbit calculations. Its definition can be written as:
$lcm(n_1, n_2)=\frac{n_1 \cdot n_2}{gc(n_1,n_2)}$ where $gc(n_1,n_2)$ is the GCD of $n_1$ and $n_2$.

To implement this function in Python, we can use the built-in `gcd()` function and an implementation of the `lcm()` function as follows:
```python
def gcd(a, b):
    while b != 0:
        y = a % b
        a = b
        b = y
    return a

def lcm(a, b):
    return (a * b) / gcd(a, b)
```
This code defines the `gcd()` function and its implementation as two nested while loops. The first loop checks if `b` is equal to zero, in which case we simply divide `a` by `b`, since its remainder will always be zero. Afterwards, we update the variables `a` and `b`. This implementation can be used with any given values of `a` and `b` as inputs, which satisfies the criteria for a LCM. We note that this implementation does not include any error handling or user input validation.
