```python
import math # for dividing two numbers modulo each other

def gcd(a, b):
    while b != 0: # find greatest common divisor
        t = b # save the remainder
        b = a % b # calculate remainder by dividing a by b
        a = t # update a with remainder a/b

    return a

def lcm(a, b):
    r = (a * b) / gcd(a, b) # calculate LCM using the formula above
    return r
```

The implementation follows the Python standard library `math` module for the division and modulo operations. The function `gcd` first divides each number by its remainder, which is then used to calculate the GCD. It then updates both variables a and b with the final quotient and remainder, respectively. The LCM formula is then calculated using the original inputs as their divisors (the final quotient), and the resulting value returned. This ensures that the LCM of two given numbers is the greatest common divisor of those two numbers, which in turn is also the least common multiple (LCM) for those two numbers.

The function should only handle inputs where both numbers are positive integers, otherwise it will return a non-LCM value and not calculate anything meaningful. The final result is then calculated using this formula and returned as a float. The `lcm` function takes in two integer arguments and returns the LCM or `None` if they don't have any common divisors or if their GCD can't be calculated due to some input error.
