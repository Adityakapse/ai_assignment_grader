To implement a seaRCh search algorithm from the lecture, we can follow the steps outlined in Point 4 of the Rubric:

1. Use Length() to calculate the length of the input list (valid range: 0–15).
2. Implement an outer loop for repeated passes and a middle-to-end loop for traversing adjacent or relevant element pairs. This loop is based on the Comparator class we have already defined, which compares each item with its corresponding item in the previous pass (or in a "step" of the current pass).
3. Increment the index of the item to consider next in the comparisons for the current pass. This can be done by calling `next()` on the Comparator object, which retrieves the next element that matches the current comparison step.
4. Use Swap() on violations of the current comparison. Swapping an element with its "older" or "newer" sibling is a valid comparison in our case and will ensure the correct order of elements.
5. The Sort method can return the sorted list once all comparisons have been performed.
