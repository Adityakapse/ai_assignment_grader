public class BinarySearch {
    public static int[] binarySearch(int[] arr, int target) {
        // If the target is not in the array or if it exceeds its length
        if (arr.length == 0 || target > arr[arr.length - 1]) {
            return null;
        }
        
        // Swap the current element and its index with the pivot element
        int pivot = Math.abs(target) % arr.length;
        int left = 0;
        int right = arr.length - 1;
        
        while (left <= right) {
            int mid = (left + right) / 2;
            
            if (arr[mid] < target) {
                left = mid + 1;
            } else {
                right = mid;
            }
        }
        
        // If the pivot index has been swapped, we found the target
        if (left == arr.length || target == arr[arr.length - 1 - left]) {
            return Arrays.copyOf(arr, left + 1);
        } else {
            int[] rightRear = binarySearch(arr, target, left, arr.length - 1 - left);
            
            if (rightRear == null) {
                return null;
            } else {
                return Arrays.copyOf(Arrays.sort(rightRear), rightRear.length + 1);
            }
        }
    }
    
    public static void main(String[] args) {
        int arr[] = {1, 3, 5, 7};
        int target = 9;
        
        System.out.println("BinarySearch: [" + Arrays.toString(arr) + "]");
        System.out.println("Target = " + target);
        
        int[] result = binarySearch(arr, target);
        
        if (result == null) {
            System.out.printf("Result is not found\n");
        } else {
            System.out.printf("Result: [%d,%d]", Arrays.toString(Arrays.sort(result))[0], Arrays.toString(Arrays.copyOf(result, result.length - 1))[0]);
        }
    }
}
```

The array is represented by an `int[]`, which has a length and a size property. We start with an empty `arr` set to be passed in as a parameter, then swap the current element and its index with the pivot element (`target` here). Swapping takes place with `Math.abs(target)` (which is guaranteed to be between 0 and arr.length) so that we don't need to keep track of a global `target`. Then we loop through the array and compare the current and pivot elements with their indices (starting from index `left` in the loop). If the comparison indicates that the pivot element is the target (and not a larger or smaller element than it), then we found it. If the pivot element is smaller, then we have found the right place to insert the target. ```java
public int binarySearch(int[] arr, int target) {
    if (arr == null || arr.length == 0) {
        return -1;
    } else if (target < arr[0]) {
        return binarySearch(arr, target);
    } else if (target > arr[arr.length-1]) {
        return binarySearch(arr, target);
    } else {
        int mid = (target - arr[0] + arr.length) / 2; // find middle element index
        if (mid <= arr.length/2 && arr[mid] == target) { // exact match found in middle
            return mid;
        } else if (arr[mid] > target) { // left sub-array
            return binarySearch(arr, target, mid+1);
        } else { // right sub-array
            return binarySearch(arr, target, arr.length-mid-1);
        }
    }
}
```

This implementation uses recursion to search for the middle element of a sorted array with equal or unequal elements based on the user's given input. It takes the target as input and returns the index of the matched element if found, else -1. The recursive function has three branches: one for the exact match at the middle index (mid), another for the left sub-array, and a third for the right sub-array depending on whether the element being searched is in the middle or not.
