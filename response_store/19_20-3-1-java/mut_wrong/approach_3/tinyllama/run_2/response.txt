Implementing the Bubble Sort algorithm from the lecture:

Here's an implementation of bubble sort that sorts a given list using a breadth-first approach to checking and swapping adjacent elements:

```csharp
public class ListToSort {
    private int[] list;

    public ListToSort(int[] input) {
        this.list = Arrays.CopyOf(input, input.Length);
    }
    
    // Implementation of BubbleSort
    public void Sort(ListToSort list) {
        
        if (list.Length() < 2) {
            return;
        }
        
        for (int I = 0; I < list.Length(); i++) {
            
            // If the element at the bottom of the list is greater than its neighbour, swap it with the neighbour
            for (int j = list.Length() - 1; j > i-1; j--) {
                if (list.Smaller(j, j-1)) {
                    int temp = list[j];
                    list[j] = list[j-1];
                    list[j-1] = temp;
                }
            }
        }
        
        return;
    }
    
    // Built-in compare function for comparing adjacent elements
    private static bool Compare(int a, int b) {
        if (a == b) {
            return false;
        } else {
            return a < b;
        }
    }
    
    public ListToSort() {}
}
```

This implementation uses a loop that checks and swaps adjacent elements in the list. The method `Compare` is used to compare adjacent elements based on their values, returning true if they are equal, false otherwise (i.e., one element is greater than or equal to its neighbour).

Note that this implementation is not strictly consistent with the student's approach, as it makes no use of the list's internal structure. However, it can be used as a starting point for implementing a more efficient bubble sort algorithm.
