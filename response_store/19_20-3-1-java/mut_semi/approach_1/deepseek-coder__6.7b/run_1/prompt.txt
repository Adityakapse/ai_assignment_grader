QUESTION:
Implement any search algorithm from the lecture
that sorts the given list.

The list has the following operations, which get as input indexes of the
list elements:

* Swap(int x, int y) <- Swaps the items at position x and y.

* Larger(int x, int y) <- Returns true if the item at
position x is greater than the item at position y

* Smaller(int x, int y) <- Returns true if the item at
position y is greater than the element at position x

* Equal(int x, int y) <- Returns true if the element at
position y is equal to the element at position x

* Length() <- Returns the length of the list

SAMPLE SOLUTION:
public ListToSort Sort(ListToSort list) {
    // Implementation of BubbleSort
    for(int i=1; i < list.Length(); i++) {
        for(int j = list.Length()-1; j > i-1; j--) {
            if (list.Smaller(j, j-1)) {
                list.Swap(j, j-1);
            }
        }
    }
    return list;
}

STUDENT SOLUTION:
// Single pass only - leaves list partially sorted
public ListToSort Sort(ListToSort list) {
    for (int j = 1; j < list.Length(); j++) {
        if (list.Smaller(j, j - 1)) {
            list.Swap(j, j - 1);
        }
    }
    return list;
}

Point 1 — Loop / iteration structure (valid range: 0–20)
  The method implements a recognisable sorting algorithm structure: an outer loop (or flag-based loop) for repeated passes, and an inner loop or mechanism for stepping through adjacent or relevant element pairs.

Point 2 — Comparison step: use of Larger / Smaller / Equal (valid range: 0–20)
  The method calls Larger(x, y), Smaller(x, y), or Equal(x, y) with the correct index arguments to detect whether two elements are in the wrong order.

Point 3 — Swap step: calling Swap on order violation (valid range: 0–20)
  When the comparison indicates that two elements are in the wrong order, Swap(x, y) is called with the same pair of indices used in the comparison.

Point 4 — Use of Length() for loop bounds (valid range: 0–15)
  Length() is used to bound all relevant loops, making the algorithm work for any list size without hardcoded lengths.

Point 5 — Overall correctness (valid range: 0–25)
  The Sort method returns the list sorted in ascending order for all valid inputs.
