To implement Bubble Sort in Java, we can use the following code:

```java
public class BubbleSort {
    public static void bubbleSort(int[] arr) {
        for (int I = 0; I < arr.length - 1; i++) {
            boolean swapped = false;
            for (int j = 0; j <= I; j++) {
                if (arr[j] > arr[j+1]) {
                    swap(arr, j, j+1);
                    swapped = true;
                }
            }
            if (!swapped) {
                break; // no more swaps needed
            }
        }
    }
    
    private static void swap(int[] arr, int i, int j) {
        int temp = arr[i];
        arr[i] = arr[j];
        arr[j] = temp;
    }
}
```

The code calls `bubbleSort` method with the given array. The method first checks if any element is swapped in any iteration (if an element moves from larger to smaller or vice versa). If not, it continues iterating and no need for more swaps. After each loop, `Swap(arr[i], arr[j])` method is called with `arr[i]` and `arr[j]`.

Both the `bubbleSort` method and `swap` methods are part of the public interface and only implement the necessary logic for sorting elements in ascending order. The rest of the implementation is done using other methods provided by Java's `Arrays` class.
