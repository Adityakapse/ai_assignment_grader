In Big-O notation, a function f(x) is said to have order O(x) if for all x ≤ a, there exists an absolute constant c such that |f(x)| ≤ ca. If this holds for all n ≥ a, then f(n) ≤ 2^a*c n. Therefore:
   |f(n)| = 2^a * |f(a)| + 2^b * |f(b)| + ... + 2^(a-1) * |f(0)|
where a is the number of iterations, and b is the number of times f(x) is applied for n > a. The maximum value in this sum (which gives the worst case scenario) is when a = b = 1: 2^1 * 2^1 + 2^0 * 2^0 = 1 + 1 = 2 (i.e., the function f(x) only gets called once).

So in Big-O notation, f(n) = O(n) if n ≤ a and |f(a)| ≤ c n. This gives us that:

- The Big-O notation of f(n) is O(n) when n ≤ a (i.e., the function f(n) gets called at most once), and the runtime of f(n) = O(n).
