public final class AsymQ5 {
    static class Node {
        int v;
        Node next;
        Node(int v, Node next) { this.v = v; this.next = next; }
    }

    static Node h;   // head of the linked list

    // Your analysis should be based on the number of nodes N in the linked list with head h.
    public static int f() {
        int s = 0;
        Node p = h;
        while (p != null) {
            s += g(p);
            p = p.next;
        }
        return s;
    }

    private static int g(Node start) {
        int c = 0;
        Node q = start;
        while (q != null) {
            c += q.v;
            q = q.next;
        }
        return c;
    }
}

SAMPLE SOLUTION:
Let N be the number of nodes in the linked list with head h.

Method f
Part A: int s = 0; Node p = h; — constant time.
Part B (while loop): traverses the list, calling g(p) at each node.
Part C: return s; — constant time.

Parts A and C are Theta(1).

Part B (loop):
- Iterations: in each iteration p advances by one node (p = p.next), so starting from h the loop runs exactly N times before p becomes null. Theta(N) iterations.
- Cost per iteration: dominated by the call to g(p). In the worst case g scans from p to the end of the list — when p = h this is N nodes, when p is the i-th node it is N - i nodes. The per-iteration cost is therefore Theta(N) in the worst case.
- Total cost of Part B: summing the actual per-iteration costs gives N + (N-1) + ... + 1 = N(N+1)/2 = Theta(N^2). Equivalently, Theta(N) iterations x Theta(N) per iteration = Theta(N^2).

Overall running time of f:
Theta(1) + Theta(N^2) + Theta(1) = Theta(N^2).

Method g
Part D: int c = 0; Node q = start; — constant time.
Part E (while loop): traverses from start until the end of the list.
Part F: return c; — constant time.

Parts D and F are Theta(1).

Part E:
- Iterations: q starts at start and advances by q.next each iteration until q becomes null. In the worst case start = h, so the loop visits all N nodes; the iteration count is Theta(N).
- Cost per iteration: Theta(1) — one += assignment, one pointer advance, one null comparison.
- Total cost of Part E: Theta(N) x Theta(1) = Theta(N).

Overall running time of g:
Theta(1) + Theta(N) + Theta(1) = Theta(N).
