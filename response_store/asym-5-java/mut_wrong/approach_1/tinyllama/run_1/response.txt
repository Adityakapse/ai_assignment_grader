Here is a brief outline of the objectives, rubrics, and analyses needed for this project. The program will be written in Java and will calculate both asymptotic best case running times for two given functions.

Objective: Calculate asymptotic worst-case runtime for both functions with given inputs.

Rubrics:
- Write a block per rubric point, then end with a total block for each rubric
- Assign a numeric mark within the valid range for each rubric point
- Write a specific analysi to cover all points in the rubric (e.g., "T_f(N)")
- Use Theta notation or Big-O notation with Tmax and Tmin bounds
- Each loop must have exactly one block, with each step given a cost per iteration
- Assign a specific analysi for each loop (e.g., "g(p)" or "h")

Analyses:
- For part A of the f() function, write a detailed analysi to cover the constant time and Theta(1) per iteration costs
- For part B of the g() function, write a detailed analysi to cover both the constant time and Theta(N^2) per iteration costs, as well as the loop total cost (e.g., "T_g(N)" or "Sum(s)")
- For both functions, write an overall worst-case runtime for all N inputs (e.g., "taken linearly, O(N)")
