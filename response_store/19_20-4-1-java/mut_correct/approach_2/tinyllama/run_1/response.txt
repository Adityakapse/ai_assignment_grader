The sub-tree with no parent should not change the input argument, but the parent will now point to the returned root node.

SAMPLE SOLUTION:
```java
public Node delete(Node root, Node node) {
    if (root == null || root != node || root.val < node.val) {
        return node;
    } else if (node.left == null && node.right == null) {
        Node x = root;
        root = null;
        return x;
    } else if (root.val > node.val) {
        root = delete(root.left, node);
        root.val = node.val;
        return root;
    } else {
        root = delete(root.right, node);
        root.val = node.val;
        return root;
    }
}
```
