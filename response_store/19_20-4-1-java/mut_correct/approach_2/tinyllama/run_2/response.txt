QUESTION:
These tasks will be about seairch trees. Our seairch trees are built from "nodes", analogous to the lecture. A node looks like this:

public class Node {
     public int key;
     public Node left;
     public Node right;
     public Node parent;
}

In the lecture you have already seen an algorithm for seairching in a seairch tree.

You are to implement different methods for the seairch trees. For this task, implement a method "insert" that insertions of a "Node" into the seairch tree and returns the new seairch tree. The nodes can be reused as desired. The function is passed the root of the seairch tree and the new node.

SAMPLE SOLUTION:
public Node insert(Node root, Node insert) {
     Node x = root;
     Node y = null; // saves Node.Parent
     while (x != null) {
         y = x;
         if (insert.key < x.key) {
             x = x.left;
         } else {
             x = x.right;
         }
     }

     insert.parent = y;
     if (y == null) {
         return insert;
     } else if (insert.key < y.key) {
         y.left = insert;
     } else {
         y.right = insert;
     }

     return root;
}

RUBRIC POINT:
Point 3 — Merge operation (valid range: 1–20)
  1. The first "Node" is taken as the root of the merged tree.
  2. The remaining nodes are traversed in order and their leaves (i.e., leaves with no parent) are inserted into the merged node's left subtree.
  3. The remaining nodes with no children are skipped and removed from the original tree.
  4. If the original tree had a null insertion point, the first root will be returned as the merged result.

public class Node {
    public int key;
    public Node left, right, parent; // saves Node.Parent
    
    public Node(int key) {
        this.key = key;
        this.parent = null;
    }
}

public Node insert(Node root, Node insert) {
    Node x = root;
    Node y = null; // saves Node.Parent
    
    while (x != null) {
        if (insert.key < x.key) {
            x = x.left;
        } else {
            x = x.right;
        }
        
        if (y == null && insert.key < y.key) {
            y = insert;
        }
    }
    
    if (y == null) { // if not found in the tree, create a new node with inserted key as its parent
        return insert;
    } else {
        insert.parent = y;
        
        if (insert.key < y.key) {
            y.left = insert;
        } else {
            y.right = insert;
        }
        
        return root; // if the node is now in the tree, update root with new parent and child nodes
    }
}
