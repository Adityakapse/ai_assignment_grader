To add comments to your code, you can use the following syntax:
```java
public Node insert(Node root, Node insert) {
    // Comment on line 10
    Node x = root;
    
    // Comment on lines 14-23
    Node y = null; // saves Node.Parent
    
    while (x != null) {
        // Comment on lines 5-8
        y = x; // comments on line 9
        
        // Comment on line 20
        if (insert.key < x.key) {
            x = x.left; // comments on lines 11-12
        } else {
            x = x.right; // comments on lines 13-14
        }
    }
    
    // Comment on line 26
    insert.parent = y; // comments on lines 15-16
    
    if (y == null) { // comments on lines 17-18
        return insert;
    } else if (insert.key < y.key) { // comments on line 20
        y.left = insert;
    } else {
        y.right = insert;
    }
    
    return root;
}
```

Note that the syntax for adding comments should follow the same style as you would use to comment lines in your code.
