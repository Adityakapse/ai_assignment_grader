```java
public Node insert(Node root, Node insert) {
    Node x = root;
    Node y = null; // saves Node.Parent
    
    while (x != null) {
        y = x; // save x if it has a parent
        
        if (insert.key < x.key) { // compare x.key with insert.key, and set y.parent to x if x.key is less than the key of insert.key
            x = x.left;
        } else {
            x = x.right;
        }
    }
    
    // if we found an empty node, set it as root (since it now contains the new node)
    if (y == null) {
        return insert;
    } else if (insert.key < y.key) {
        y.left = insert; // set left child of parent to the newly inserted node
    } else {
        y.right = insert; // set right child of parent to the newly inserted node
    }
    
    return root;
}
```

In this implementation, we first check whether x is null (which means that it hasn't been created yet), and if so, we create a new Node using the "new" keyword with the appropriate values for x.parent (if it has no parent) or y.parent (if it already exists as a child of its parent). Then we set the corresponding fields of the new node to match the input data.

After finding the root and inserting the new node, we return the root node (since we don't need to traverse further to insert the child nodes). The "while" loop has a conditional statement that checks if x is null before entering the loop. If it's null, we set x to the first Node in the tree and start the recursion for x.
