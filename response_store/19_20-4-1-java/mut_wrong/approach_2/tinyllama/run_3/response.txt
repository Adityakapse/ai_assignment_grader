You can see the implementation of insert() method in our seaRCch tree, which is given below:

```
public Node insert(Node root, Node insert) {
     Node x = root;
     Node y = null; // saves Node.Parent
     while (x != null) {
         y = x;
         if (insert.key < x.key) {
             x = x.left;
         } else {
             x = x.right;
         }
     }

     insert.parent = y;
     if (y == null) {
         return insert;
     } else if (insert.key < y.key) {
         y.left = insert;
     } else {
         y.right = insert;
     }

     return root;
}
```

The function "insert" has a loop that iterates through the tree and checks if the new node can be inserted into any of its children, and moves it to the proper parent if necessary. If the new node is not currently in the tree, it is inserted into its parent node. 

Inside the loop, we compare the current root with the new insertion point (x), which has the same key as the current node found during traversal (y). If the keys are equal or greater than each other, then the new node is inserted in its proper parent node. 

If the key of the current node is less than that of the insertion point (x), we move the current node to be its parent, i.e., x.parent = y; statement is executed, and the loop continues with the parent node. If the keys are greater than each other, then the new node is inserted in its proper child node, by swapping y with its left child (x.left) and x with its right child (y.right), respectively.

Finally, we return the root node of the tree that has been modified.
