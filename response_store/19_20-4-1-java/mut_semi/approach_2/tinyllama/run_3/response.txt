Implement a Method for Insertion in SeaRChe Trees:
The following code demonstrates how to implement the insertion method in seaRChe trees. We first check that the root is not null, then traverse through the tree until we find the appropriate position (insertion point) by comparing the key of the current node with the insertion key and moving accordingly using a loop or recursive function call.

public Node insert(Node root, Node insert) {
     Node x = root;
     Node y = null; // saves Node.Parent
     while (x != null) {
         y = x;
         if (insert.key < x.key) {
             x = x.left;
         } else {
             x = x.right;
         }
     }

     insert.parent = y; // set inserted node's parent to its correct parent in the tree
     if (y == null) { // base case: if the inserted node is not found, we are done here
         return insert;
     } else if (insert.key < y.key) { // move the inserted node to the left of its matching parent in the tree
         y.left = insert;
     } else { // move it to the right of its matching parent
         y.right = insert;
     }

     return root;
}
