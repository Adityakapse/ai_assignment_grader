```java
import java.util.Stack;

public class IntegeerStack {
    private int head = -1; // The index of the current element being processed
    private int empty = 0; // The index of the first element on the stack that is not an op or a number
    
    public boolean isEmpty() { return head == -1; }
    
    public void push(int i) {
        if (isEmpty()) throw new Error("Stack underflow"); // If the stack has no elements, we throw an error
        empty = i;
        head++;
    }
    
    public int pop() {
        while (!isEmpty()) pop();
        if (head == -1) throw new Error("Stack overflow"); // Check if the stack is full
        return empty--;
    }
    
    public String toString() {
        StringBuilder sb = new StringBuilder(empty);
        while (!isEmpty()) {
            sb.append(pop());
        }
        return sb.toString();
    }
    
    // Other methods omitted for brevity
}
```

This implementation assumes that the input is a valid formula, and does not validate any of the operators or parentheses. The `isEmpty` method checks if the stack is empty, while the `pop` method removes the element at the top of the stack. The `toString` method converts the Stack to a String for ease of use.

1. Define an interface for a calculator:
   ```java
   public interface IntegerStack {
       void push(int i);
       int pop();
       int head();
   }
   ```

2. Implement the Stack class:
   ```java
   public class IntegerStack implements IntegerStack {
       private int[] top; // pointer to top element of stack
       private int size; // number of elements in stack

       public void push(int i) {
           if (size == top.length) { // grow stack
               int[] newTop = new int[top.length * 2];
               System.arraycopy(top, 0, newTop, 1, size); // copy old elements to new stack
               top = newTop; // set new top pointer to new stack
           }
           top[size] = i; // insert element at front of stack
           size++; // increment size counter
       }

       public int pop() {
           if (size == 0) {
               throw new EmptyStackException();
           }
           int value = top[--size]; // remove element from top of stack
           if (size == 1) { // if top is empty, return top element (which is the result)
               return value;
           }
           pop(); // remove remaining elements, continue as before
       }

       public int head() {
           return top[0]; // return frontmost element of stack
       }
   }
   ```

3. Implement the Calculator class:
   ```java
   public class Calculator implements IntegerStack {
       private String[] input; // array containing formula as string, including operators and decimal points
       private IntegerStack s; // stack used to execute operator by pop(int) / push(int)

       public int calculate() {
           for (String oper : input) {
               switch (oper.charAt(0)) {
                   case '+':
                       int result = s.pop(); // pop top element (result) from stack and assign it to variable
                       if (s.emptystack()) { // if stack is empty, throw EmptyStackException
                           throw new EmptyStackException();
                       } else {
                           s.push(result); // push result back onto stack
                       }
                       break;
                   case '-':
                       int result = s.pop(); // pop top element (result) from stack and assign it to variable
                       if (s.emptystack()) {
                           throw new EmptyStackException();
                       } else {
                           s.push(result * -1); // multiply result by -1 and push back onto stack
                       }
                       break;
                   case '*':
                       int result = s.pop(); // pop top element (result) from stack and assign it to variable
                       if (s.emptystack()) {
                           throw new EmptyStackException();
                       } else {
                           s.push(result / 1); // divide result by 1 and push back onto stack
                       }
                       break;
                   case '/':
                       int result = s.pop(); // pop top element (result) from stack and assign it to variable
                       if (s.emptystack()) {
                           throw new EmptyStackException();
                       } else {
                           int remainder = s.pop(); // get the remainder after dividing result by 1
                           s.push(result * remainder); // multiply result with remainder and push back onto stack
                       }
                       break;
                   default:
                       s.push(Integer.parseInt(oper)); // interpret operator token as a number, push it onto stack
               }
           }
           return s.pop(); // return last element of the stack (result), pop it and return
       }
   }
   ```
