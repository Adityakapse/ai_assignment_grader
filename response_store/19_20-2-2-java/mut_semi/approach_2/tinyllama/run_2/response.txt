public class IntegeRStack {
    public boolean isEmpty() {
        return head == -1;
    }
    
    public int head() {
        if (isEmpty()) {
            throw new RuntimeException("Stack is empty");
        } else {
            return top();
        }
    }
    
    public void push(int i) {
        if (isFull()) {
            throw new RuntimeException("Stack is full");
        }
        int newTop = head;
        while (!isEmpty() && top() == head) {
            --top;
        }
        while (!isEmpty() && top() != head) {
            top--;
        }
        if (newTop < 0) {
            throw new RuntimeException("Stack underflow");
        } else {
            top = newTop;
        }
    }
    
    public int pop() {
        if (!isEmpty()) {
            int result = top;
            top = top - 1;
            return result;
        } else {
            throw new RuntimeException("Stack is empty");
        }
    }
    
    private boolean isFull() {
        return head == (size() - 1);
    }
    
    public int size() {
        if (!isEmpty()) {
            int result = top;
            while (!isEmpty()) {
                top--;
            }
            return result + 1;
        } else {
            throw new RuntimeException("Stack is empty");
        }
    }

If a number is processed, it is taken off the stack, and if an operator is encountered, its upper two numbers (the operands) are popped from the stack. If there's an operator in between two numbers, it's applied, and then the result is pushed back on the stack. After the calculation, the result is the only number on the stack.
