Exercise:
Write a C++ program using Queue and Queue.jl that allows the user to create multiple instances of the Queue class, and then perform various operations on them. The program should display a list of all created instances at the end of execution. The operations should include adding, removing, and displaying elements in a sorted or unsorted manner. Additionally, the program should handle invalid input (such as empty arrays) gracefully. The code should use appropriate naming conventions, comments, and be well-documented. Finally, the program should be tested using various inputs to ensure correct functionality.
