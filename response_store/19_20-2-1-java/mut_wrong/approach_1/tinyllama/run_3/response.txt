Class "Queue" implements a simple stack, where each element in the array is represented as a single string. The class also contains methods for emptying the stack, retrieving the head of the queue (i.e., the most recently added element), and removing the front (i.e., the last element) of the queue.

The head and tail indices, respectively, represent the index of the first and last elements in the buffer. All methods work correctly, and are valid within their respective ranges of input (i.e., they do not modify any internal state).

To implement an empty queue, the Queue class simply returns true if and only if it contains no elements (valid range: 0–15). The head() method returns the front element without modifying the internal state, while enqueue(x) inserts the given string into the buffer array at the appropriate position.

To implement a dequeue operation, the Queue class retrieves the element at the front of the queue (valid range: 0–18), removes it from the buffer array, and updates all internal tracking state accordingly. The dequeue() method returns the last element retrieved (i.e., the head of the queue) without modifying any state.
